Real-time fall detection pipeline built on YOLOv5 for image, video, webcam, and edge-device inference.

## Highlights

- Supports both local inference and remote deployment workflows, including PyTorch, ONNX, and Atlas200DK `.om` models.
- Exposes command-line and Python API entry points for image, video, and live camera fall detection.
- Packages trained weights and remote inference tooling into one repo so the same project can be used for experimentation and deployment.
